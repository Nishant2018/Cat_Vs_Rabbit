Cat vs Rabbit Classifier

Welcome to the Cat vs Rabbit Classifier! This project showcases a machine learning model designed to identify and classify images of cats and rabbits. The model has been trained on a diverse dataset, capturing various poses, lighting conditions, and backgrounds for both cats and rabbits.

Project Highlights:

  • Machine Learning Model: The classifier is built using a Convolutional Neural Network (CNN), a type of deep learning model known for its effectiveness in image recognition tasks.
  • Dataset: The model has been trained on a carefully curated dataset consisting of thousands of cat and rabbit images. This dataset ensures the model's ability to generalize well to new and unseen images.
  • Prediction Confidence: Alongside predictions, the classifier provides a confidence score. A higher confidence score indicates that the model is more certain about its prediction.
  • User Interaction: Use the "Choose File" button to upload an image, and click "Predict" to observe the model's prediction. The result, along with the confidence score, will be displayed below.
  • Instructions: Follow these steps for a prediction:
    1. Click the "Choose File" button to select an image file (JPEG or PNG).
    2. Click the "Predict" button to see the classifier's prediction.
    3. The result will be displayed below the buttons, indicating whether the image is predicted to be a cat or a rabbit, along with the confidence score.

Dataset Information:

The dataset used for training and testing consists of a diverse collection of cat and rabbit images. It helps the model generalize well to various poses, lighting conditions, and backgrounds.
